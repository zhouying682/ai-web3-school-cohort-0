# 技术驱动型商业人物：留学与职业路径分析

> 核心结论：方向不是"找一份普通高薪工作"，而是**进入未来技术浪潮中心，为创业做准备**。

## 偶像共性：技术变革最前线

- **Elon Musk**、**Jensen Huang**、**Lei Jun**
- 三人共同点：都站在**技术变革最前线**，而非传统职业路径

## 什么环境最容易培养创业机会

1. 技术密度 — 身边到处是工程师和 builder
2. AI 生态 — agents、LLM、robotics、AI infra
3. 高质量同学 — 聊 startup、开源、prototype，而非考证
4. 实验室/项目 — 从 side project 中学习
5. 创新氛围 — 不满足于"找份好工作"

## BUILDER 特征

- 长期研究趋势 → 对 AI 有真实兴趣 → 不满足于纯商科 → 想做创造者
- 这很像**技术创业者早期人格**

## 创业最怕什么：进入错误生态

| 错误生态 | 正确生态 |
|---|---|
| 考证、找稳定工作 | agents、robotics |
| 咨询、银行 | AI infra、LLM |
| PPT、case interview | startup、prototypes |

> 环境决定了你往哪个方向推

## 新加坡 vs 澳洲：AI 创业生态

- 新加坡：政府疯狂投 AI、国际资本聚集、AI 产业链强、亚洲 AI 试验场
- 澳洲：AI 生态分散、创业资本密度低、很多人最终走传统就业

## 四校排序

1. **SUTD** — AI 氛围 + Build culture + 创新生态
2. **SMU MITB** — 商业 AI + networking，偏职业经理人
3. **Adelaide** — AIML 研究强，生态不如新加坡
4. **UWA** — 偏传统综合大学

## 现在该积累什么

- ❌ 不要执着：title、GPA、证书
- ✅ 应该积累：Coding、AI 理解、**做项目**（side project 是 AI 创业者起点）

## 关键判断

你有**技术理想主义**。如果进入纯传统职业路径，几年后容易内耗：
> "我每天到底在干什么？"

**结论：SUTD → 高密度 AI 环境 → Build 能力 → 技术创业。**
